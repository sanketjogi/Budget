# 💰 BudgetFlow — Personal Budget Tracker

A premium, Apple-inspired budget tracker built with vanilla HTML, CSS & JavaScript. Features a stunning **liquid glass UI** that automatically matches your system's light or dark theme.

![BudgetFlow Dashboard](https://img.shields.io/badge/Status-Live-brightgreen) ![HTML](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white) ![CSS](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

## ✨ Features

- **Apple Liquid Glass Design** — Frosted glass cards, animated ambient blobs, smooth spring animations
- **Auto Light/Dark Mode** — Seamlessly matches your system theme via `prefers-color-scheme`
- **Dashboard** — Total balance, monthly income/expenses with animated counters
- **Interactive Charts** — Spending breakdown (donut), monthly trends (bar), category comparison
- **Transaction History** — Full searchable, filterable history grouped by date
- **Smart Insights** — Personalized tips based on your spending patterns
- **14 Expense Categories** — Including 🚬 Smoking & 🍺 Alcohol tracking
- **Mobile Responsive** — Beautiful bottom navigation bar on smaller screens
- **Local Storage** — Data persists across sessions, no server needed
- **Toast Notifications** — Smooth feedback on every action
- **Currency** — ₹ INR

## 🚀 Getting Started

Simply open `index.html` in your browser — no build tools required!

Or serve locally for the best experience:
```bash
npx serve .
```
